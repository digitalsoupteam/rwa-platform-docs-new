# Contract Reference

This page documents all smart contracts in the Slices protocol, their key functions, and access control.

## System Contracts

### AddressBook

The central registry. Stores addresses of all protocol contracts and manages access control.

| Function | Access | Description |
|---|---|---|
| `setGovernance(Governance)` | Governance | Update governance contract |
| `setConfig(Config)` | Governance | Update config contract |
| `setFactory(Factory)` | Governance | Update factory contract |
| `setTimelock(Timelock)` | Governance | Update timelock contract |
| `setTreasury(Treasury)` | Governance | Update treasury contract |
| `setDaoStaking(DaoStaking)` | Governance | Update DAO staking contract |
| `setPlatformToken(PlatformToken)` | Governance | Update platform token contract |
| `setReferralTreasury(ReferralTreasury)` | Governance | Update referral treasury |
| `setEventEmitter(EventEmitter)` | Governance | Update event emitter |
| `setRWAImplementation(address)` | Governance | Update RWA implementation |
| `setPoolImplementation(address)` | Governance | Update pool implementation |
| `addSigner(address)` | Governance | Add authorized signer |
| `removeSigner(address)` | Governance | Remove authorized signer |
| `addPool(Pool)` | Factory | Register a pool |
| `addRWA(RWA)` | Factory | Register an RWA token |
| `getPoolByIndex(uint256)` | Anyone | Get pool by index |
| `getRWAByIndex(uint256)` | Anyone | Get RWA by index |
| `rwasLength()` | Anyone | Number of registered RWAs |
| `poolsLength()` | Anyone | Number of registered pools |

## RWA Contracts

### Factory

Deploys RWA token and Pool contracts via ERC1967 proxy.

| Function | Access | Description |
|---|---|---|
| `deployRWA(uint256,string,string,string,address,address[],bytes[],uint256)` | Anyone + signatures | Deploy RWA token |
| `deployPool(uint256,string,string,string,address,RWA,uint256,uint256,uint256,uint256,uint256,uint256,uint256,uint256,uint256,bool,bool,bool,bool,uint256[],uint256[],uint256[],uint256[],address[],bytes[],uint256)` | Anyone + signatures | Deploy pool |

### RWA

ERC1155 multi-token contract for real-world assets.

| Function | Access | Description |
|---|---|---|
| `createToken(address)` | Factory | Create new token ID |
| `mint(address,uint256,uint256)` | Associated pool | Mint tokens |
| `burn(address,uint256,uint256)` | Associated pool | Burn tokens |
| `uri(uint256)` | Anyone | Get metadata URI |
| `enablePause()` | Governance | Pause contract |
| `disablePause()` | Timelock | Unpause contract |

### Pool

Crowdfunding pool with AMM and tranches.

| Function | Access | Description |
|---|---|---|
| `mint(uint256,uint256,uint256,bool)` | Anyone | Buy RWA tokens |
| `burn(uint256,uint256,uint256,uint256)` | Anyone | Sell RWA tokens |
| `claimOutgoingTranches(uint256[])` | Pool owner | Claim outgoing tranches |
| `returnIncomingTranche(uint256)` | Anyone | Return funds |
| `estimateMint(uint256,bool)` | Anyone | Estimate mint cost |
| `estimateBurn(uint256)` | Anyone | Estimate burn return |
| `enablePause()` | Governance | Pause pool |
| `disablePause()` | Governance | Unpause pool |
| `getOutgoingTranches()` | Anyone | Get outgoing tranche data |
| `getIncomingTranches()` | Anyone | Get incoming tranche data |
| `getBonusUnlockTimestamp()` | Anyone | Get bonus unlock time |
| `checkBonusesUnlocked()` | Anyone | Check if bonuses available |

## DAO Contracts

### Governance

Manages proposals and voting.

| Function | Access | Description |
|---|---|---|
| `propose(address,bytes,string)` | Voting power ≥ threshold | Create proposal |
| `vote(uint256,bool,string)` | Staker with voting power | Vote on proposal |
| `cancel(uint256)` | Proposer (≤12h) or contract | Cancel proposal |
| `getProposal(uint256)` | Anyone | Get proposal details |
| `getReceipt(uint256,address)` | Anyone | Get vote receipt |
| `isActive(uint256)` | Anyone | Check if proposal active |

### Timelock

Enforces time delay on governance actions.

| Function | Access | Description |
|---|---|---|
| `queueTransaction(address,bytes,uint256)` | Governance | Queue transaction |
| `executeTransaction(address,bytes,uint256)` | Anyone | Execute queued transaction |
| `cancelTransaction(address,bytes,uint256)` | Governance | Cancel queued transaction |
| `withdrawERC20(address,address,uint256)` | Timelock | Withdraw ERC20 |
| `withdrawETH(address,uint256)` | Timelock | Withdraw ETH |

### Treasury

Holds protocol fees.

| Function | Access | Description |
|---|---|---|
| `withdrawERC20(address,address,uint256)` | Timelock | Withdraw ERC20 tokens |
| `withdrawETH(address,uint256)` | Timelock | Withdraw ETH |
| `getTokenBalance(address)` | Anyone | Get token balance |

### DaoStaking

Manages PlatformToken staking for governance.

| Function | Access | Description |
|---|---|---|
| `stake(uint256)` | Anyone | Stake tokens |
| `unstake(uint256)` | Anyone | Unstake tokens |
| `lock(address,uint256)` | Governance | Lock user's stake |
| `calculatePendingRewards(address)` | Anyone | Calculate pending rewards |
| `getAvailableRewards()` | Anyone | Get available rewards |
| `getVotingPower(address)` | Anyone | Get voting power |
| `getTotalVotingPower()` | Anyone | Get total voting power |

## Platform Contracts

### PlatformToken

Upgradeable ERC20 for staking and governance.

| Function | Access | Description |
|---|---|---|
| `mint(address[],uint256[])` | Governance | Mint tokens to multiple addresses |

### ReferralTreasury

Manages referral rewards with multi-signature withdrawal.

| Function | Access | Description |
|---|---|---|
| `withdraw(address,uint256,uint256,address[],bytes[])` | Anyone + signatures | Withdraw rewards |
| `emergencyWithdraw(address,address,uint256)` | Governance | Emergency withdrawal |

## UpgradeableContract

Base contract for all upgradeable contracts (UUPS pattern).

| Function | Access | Description |
|---|---|---|
| `upgradeTo(address)` | UpgradeRole | Upgrade implementation |
| `upgradeToAndCall(address,bytes)` | UpgradeRole | Upgrade and call |
| `uniqueContractId()` | Anyone | Get contract identifier |
| `implementationVersion()` | Anyone | Get version number |
